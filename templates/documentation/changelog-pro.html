Version 13 (June, 2023)

(Internal Version 1.13.2)

This version brings new updates and bug fixes to the Atlas Pro theme. In this update we focus on some fixes and small improvements. You may have noticed Atlas Pro updates slow down, as we've been focusing on developing exciting new products for the Template Marketplace. We'll be returning to more regular, bi-weekly updates starting in July.

General Updates and Bug Fixes

  • We've disabled the Animate On Scroll functionality inside the drag and drop editor preview, as this was causing issues for some users. We also added an option in the theme settings to disable Animate on Scroll altogether, this is helpful for troubleshooting issues or improving page loading performance.
  • We fixed a bug causing the footer link hover color setting to not work as expected.

Version 1.6.5 (June 2023)

(HubSpot version 6)

This version brings new updates and bug fixes to the Atlas Pro theme. This new version mainly focuses on the theme being up to date with the new HubSpot Marketplace theme requirements, fixes to bugs and errors, and new features and modules.

New Modules

  • Added a new module: Content Toggle. This module allows content in the page to be toggled between 2 different states. For example, a pricing page can now include a monthly and a yearly pricing in the same page, with a toggle to change between them. Please check this knowledgebase article for a step-by-step guide on how to use this new module: Link to Article
  • Added a new module: Global Blog Post Before Cards Text. This module allows the text before the "Similar Posts" section on the blog posts to be updated with any text now. It was hardcoded before.

New Features

  • Added a new Drag and Drop section to the global header. You can now select between 3 different options for the header. Normal, Mega Menu or full Drag and Drop.
  • Added a new Drag and Drop section to the global footer. You can now select between 3 different options for the footer. Normal, Simple or full Drag and Drop.
  • The Blog Tags module (used to insert the list of tags in the blog listing page) includes a new field to change the first "ALL" tag text. It allows the text to be updated with any text now. It was hardcoded before.

General Updates and Bug Fixes

  • Updated the blog-index.html template (Legacy Blog Listing template, non drag and drop) to include a drag and drop area at the bottom. Legacy users that are still using this template should not have any problems with it. New users should use the updated blog-index-drag-n-drop.html template instead.

Version: 1.3.7 (April 2022)

(HubSpot version 3)

This version brings new updates and bug fixes to the Atlas Pro theme. The main objective of this update is to add new features for the mobile templates and views.

Updated: April 8, 2022

We are aware that our newest update to the Atlas theme introduced some issues and unexpected behavior. Features introduced in this update will help Atlas to be more usable, robust and stable in the future as we try to keep up with the latest tools that Hubspot provides for themes and templates. However, due to the nature of HubSpot CMS, changes like these are unavoidable.
We are really sorry for any inconvenience, but know that we are working closely with HubSpot right now to better understand the specifics and prevent situations like this moving forward.
We sincerely apologize. We know it's really frustrating when you are being negatively affected by any update. We're constantly learning how to better deliver and support high-value products for the HubSpot Community. Unfortunately, it sometimes comes at the cost of issues like these. We've taken many learnings from this situation to heart and will adopt new processes into our development cycle to prevent issues like this from happening in the future.
This will only affect users that were using an older version of Atlas and updated their theme to the new version.
New users (for both Atlas and Atlas Pro) WILL NOT be affected by these issues.
Affected modules: (will update as we have more information)

  • Navbar with Menu - Hovering the navbar items doesn't display the megamenu
  • Navbar with Menu - Anchoring items to the same page causes the items to be highlighted
  • Simple Navbar - Anchoring items to the same page causes the items to be highlighted
  • Global Headers (Navbar with Menu and Simple Navbar) need to be rebuilt and reconfigured.
  • Global Footers (Simple Footer and Footer) need to be rebuilt and reconfigured.

Please don't hesitate to reach if you have additional questions or concerns.

New Modules

  • Added a new module: "Testimonial Slider"
  • Added a new module: "Image + Text"
  • Added a new module: "Callout"
  • Added a new module: "Product Features 04"
  • Added a new module: "Blog Posts" (Mainly used on the new drag and drop blog listing template)
  • Added a new module: "Blog Tags" (Mainly used on the new drag and drop blog listing template)

New Features

  • Added support for mobile templates. This allows Atlas users to hide and show specific sections only for desktop or mobile. You can learn more about this here
  • Added a new setting on the Global Theme Settings to control responsive typography
  • Added a new setting on the Global Theme Settings to hide/unhide the blog summary meta description on the blog post page template
  • Added a new blog post template with a different structure than the first one. (The old blog post template is still available).
  • Added a new drag and drop template for the blog listing template. (The old non drag and drop template is still available). This also means that a new Blog Post module and a new Blog Tags module were also generated.
  • Added a new Tooltips class. This class allows you to create small helper cards with text that can be inserted anywhere.

General Updates and Bug Fixes

  • Fixed an issue where the Hubspot Chat would be located under the header on mobile (making it unusable)
  • Fixed an issue when using the "Navbar with Menu" module where previous menus would not close if hovering over an option that didn't have a menu
  • Fixed an issue where the "Offer Bar 01" module would not be updated with the global theme settings for the cards
  • Fixed an issue where the footer image would not be left aligned
  • Fixed an issue where using internal Hubspot CTAs would generate a double wrapper around the button
  • Fixed an issue where using internal Hubspot CTAs would not make the whole button clickable, but only the text
  • Fixed an issue with multiple instances of the Product Features 03 module on the same page would cause it to break
  • Removed all styling sheets from all the modules. This will not affect any installed instances of Atlas nor will it break any styling. The purpose of this removal is to make it easier to create child themes from Atlas in the future by eliminating module specific stylesheets (This will make overwriting child themes a lot faster and easier).
